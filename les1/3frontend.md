# Frontend map aanmaken

Het doel van dit hoofdstuk is
* Het maken van een frontend project via vite

## Vite
We hebben nu een map aangemaakt voor onze backend en docker. Nu moeten we een folder aanmaken voor onze frontend. Dit doen we vanaf nu met **vite**.

Open de root folder van jouw project in de commandline/terminal

`npm create vite@latest`

Maak een vanilla javascript project aan.
