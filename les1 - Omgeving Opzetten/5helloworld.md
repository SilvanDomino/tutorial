# Hello world (PHP)
Dit hoofdstuk staat in het teken van zorgen dat de PHP werkt.
* We gaan eerst kijken of onze PHP draait zoals het zou moeten draaien
* Daarna gaan we kijken of onze API route werkt.

Maak een 'hello world' in jouw index.php om te kijken of de code uitgevoerd wordt. 

## API

Maak in de map `/php/api` een bestand aan genaamd **todos.php**. Dit bestand is de route waarmee we onze todo's gaan ophalen.
Laat deze een json array met string returnen. Probeer uit. Maak in je browser een call naar [https://127.0.0.1:8080/api/todos.php](http://localhost:8080/api/todos.php)

We hebben nu een werkende project structuur!
