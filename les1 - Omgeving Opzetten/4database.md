# Database
Het doel van dit hoofdstuk is
* Het maken van een table in onze database
* Het vullen van onze database

Onze Todo's komen in de database te staan. Daarvoor moeten we een database aanmaken, en elke todo heeft een paar eigenschappen die ook in de table moeten.

* id (uniek id, hoort in elke db)
* text (De tekst die bij de todo hoort. bv "boodschappen doen")
* status (bijvoorbeeld of de todo al af is, of gearchiveerd)
* timestamp (wanneer is de todo gemaakt)

## Seed de database
Vul de database met een paar todo's. Zodra de database gevuld is kan je deze gebruiken.
